# بوت Telegram — الشركة الهندسية التقدمية

## تشغيل البوت محلياً
```bash
pip install -r requirements.txt
python bot.py
```

## رفع على Railway (مجاني)
1. روح على railway.app
2. New Project → Deploy from GitHub
3. اختر هاد الـ repo
4. Root Directory: telegram-bot
5. أضف Environment Variable: TOKEN = [token البوت]

## المميزات
- عرض الباقات والأسعار
- حاسبة تكاليف بالدولار والليرة السورية
- المتجر الاختياري
- تواصل مباشر عبر واتساب
